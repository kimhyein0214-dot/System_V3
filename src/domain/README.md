# Domain

Standard internal names live here.

Examples:

- `receiptDate`: Sellpia receipt/collection date, current DB `ord_date`
- `invoiceNo`: current DB `inv_no`
- `sellpiaOrderNo`: current DB `ord_no`
- `sellpiaItemNo`: current DB `item_no`
- `sellpiaProductCode`: current DB `order_items.p_code`
- `ownCode`: current DB `prod_code`, `p_dpcode`
- `shortageQty`: current DB `short_qty`, `shortage_qty`
- `drawerMemo`: current DB `drawer_no`
